# wekan-deb
Debian Package for Wekan

## Supported operating systems

* [Debian Wheezy 7 64 bit](https://www.debian.org/)
* [Devuan Jessie 1 64 bit](https://devuan.org/)
* [Devuan Ascii 2 64 bit](https://devuan.org/)
